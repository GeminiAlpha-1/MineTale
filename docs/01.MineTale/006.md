::: timeline 2026-01-21
- Minecraft 推出全新萌系幼年坐骑。
- Minecraft Java 版快照和 Bedrock 预览版现已上线五种重新设计的幼年坐骑（马、驴、骡子、骷髅马、僵尸马），拥有更小巧可爱的模型和专属音效，邀请玩家测试并反馈。
- 原文链接：https://www.minecraft.net/zh-hans/article/minecraft-26-1-snapshot-4
:::

::: timeline 2026-01-23
- Simon回应Jebs对Hytale的评价。
- Hytale创始人Simon感谢Mojang的Jens Bergensten（Jebs）试玩并给予反馈，强调Hytale旨在打造独特的奇幻冒险体验，而非与Minecraft竞争或取代它。
- 原文链接：https://x.com/Simon_Hypixel/status/2014377275035115861?s=20
:::
