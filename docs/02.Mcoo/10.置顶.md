<span style="font-weight: bold; font-style: italic; font-size: 20px; background: linear-gradient(to right, rgb(252, 92, 125), rgb(106, 130, 251)); -webkit-background-clip: text; background-clip: text; color: transparent;">DoubleSpirit121——</span>

<span style="font-weight: bold; font-size: 18px;">双灵，</span>

缘分始于2018年的9月。

结束了中考，闲来无事的他决定去B站上找一些原版生存服务器玩。

在B站的首页上，他看到了明月庄主的粉丝服，便顺手点了进去，复制IP，启动游戏，进入服务器，一气呵成。

于是，他结识了戈登、电童、夕暮等一批服务器玩家。

而后又在庄主内服结识了北极熊等玩家。

玩家一批批的来，一批批的走。最终留下的还是那批人。

退出庄服后，他便开始运营起自己的服务器。

时间过得很快，服务器也换了很多周目。但是成员始终都是那批人。

为什么这么想开一个完全属于他自己的服务器呢？

因为从他接触到Minecraft多人游戏的那一刻起，便有了开一个完全属于自己的服务器的想法，这个想法随着时间的流逝越来越清晰、强烈。

浅浅的纪念一下2013年他与朋友一起开设的第一台Minecraft PE服务器。

自从中考完以后终于MC入正补票，他总想找一个正版服务器玩，于是他找到了庄主的粉丝服，砸了亿点钱，混了个管理当当……

然后事情发展如上文所述 (DoubleSpirit121 et al., 2025)

<span style="font-weight: bold; font-style: italic; font-size: 20px; background: linear-gradient(to right, rgb(252, 92, 125), rgb(106, 130, 251)); -webkit-background-clip: text; background-clip: text; color: transparent;">RSDPolarbear——</span>

<span style="font-weight: bold; font-size: 18px;">北极熊，</span>

一切，还要从那次疫情说起......

还记得，那是我中考完的第一个寒假，我正处在疫情的中心——武汉，封城、停课，完全无法出门，不过我还算呆的住，毕竟本来就是个死宅，那段时间，我更是每天除了上课就是补番，反倒是给我找了个不用出门的好理由。

直到有一天，想起自己有段时间没玩MC了，而且之前一直玩的是盗版的PE和PC，于是也是研究了入正，之后在B站一翻，最先引起我的注意的就是明月庄主的粉丝服，我没有多想直接就申请进服了，就是在这里，结识了双子，戈登这些朋友们。

明月庄主的教程算是我的红石启蒙，我对庄主的印象一直停留在了2015年，他或许不是最牛逼的大佬，但是他绝对是早期大多数玩家的红石引路人，当时受庄主影响，我一直主要在玩械电和模电，对于各种生电机制还只是一个小萌新，只是知道一些最基本的东西。

而这时，就不得不提到2020年的另一个大事件，借着全民居家的时代，TIS生电服的一周年视频，将原版生电推入了大众视野，我也是在这时，系统的学习了很多生电知识，后来进入庄服内服，开始系统的玩一些生电内容。

后来我们几个朋友因为一些原因，自己出来搭建了服务器，之后又走过了这么久，一晃已经5年了，我甚至在现实中没有一个朋友是能和你们一样高强度保持5年的联系的，我是2012年开始第一次接触MC，没想到，这款游戏能陪我走过这么远，也是借着这个游戏，让我学会了很多的东西。

玩家来来回回最后留下了这么一批人，希望可以一直走下去。

说好要一起组一辈子服务哇（笑）

<span style="font-weight: bold; font-style: italic; font-size: 20px; background: linear-gradient(to right, rgb(252, 92, 125), rgb(106, 130, 251)); -webkit-background-clip: text; background-clip: text; color: transparent;">Y.K.Suzumiya——</span>

<span style="font-weight: bold; font-size: 18px;">凉宫山风，</span>

献给此刻迈步向前的你：

2020年，你在机缘巧合下认识了生电玩法，那个时候的你看到这种以前完全没有见过的玩法唤醒了自己尘封的记忆，那段缘于你学生时期和同学们拿手机玩我的世界的日子，想起了那个早已经没有玩下去的游戏，你想去重拾它，纵使你从未接触过JAVA版本的我的世界，甚至其实完全不会红石技术，但是你依旧想去尝试挑战学习一下。

之后，你看到了珍珠炮宣传视频，你被这种恐怖的红石机件震撼到了，同时在机缘巧合下你认识了一个叫做凉音的朋友，纵使你们之间最终闹得不欢而散，但是拜凉音所赐你加入了你人生当中第一个生电服务器——WT，你隐藏了你其实对红石一窍不通的秘密，和服务器内的众人一起做工程、一起玩游戏、一起在机器上跳来跳去，在这里，你认识了一位真正带你踏上了红石玩法的朋友、也是相识五年直至今日的挚友——北极熊RSDPolarbear。

在这之后，因为某些原因WT结束了运营、玩家们分崩离析，北极熊联系了你，说他打算和他的朋友们成立一个服务器，问你要不要参加，你答应了，于是你在你的技术依旧不佳的时候，加入了这个名叫Alioth-DWS/GeDK的服务器，认识了许多、也会相伴很久的朋友，你的故事正式开启了篇章。

依旧是在这一年，你刷到了一条视频，一条来自reberd小红的红石管风琴的演奏视频，你看到了一种与以往的红石音乐完全不同的表现形式，你想要根据这个形式设计一台属于自己的编码播放器，Alioth-DWS/GEDK的创造服提供了这种便利，但是谁都未曾想到，这个目标你坚持了五年，慢慢的学习，一步步的学习着那些红石的基础理论，怎么走线、如何调整时序等等的技巧，你开始能初步设计出一些机器，能够实现一些你自己的想法。

然后，一年又一年，你最终在这个服务器的延续—Alioth-DWS/MCOO中实现了你最初的梦想，你的机器成功设计落地，你也从一个对红石完全不了解的小白慢慢变成了一个拥有一定技术的玩家，同时，你也拥有了一批和自己相伴五年的挚友。你总是说这台编码红石音乐播放器是你五年磨一剑，但是你自己何尝不是在这五年中不断磨砺自己，虽然你还不是我的世界中优秀的红石玩家，但是相信你纵使在完成这个梦想之后，依旧会不断的、继续的走下去。

这段文字，献给每一个引领你的创作者、献给你所认识的挚友们、献给Alioth-DWS服务器，最后，也献给此刻迈步向前的你。
